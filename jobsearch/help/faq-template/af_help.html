  • Applicant Search Parameters

  • Part I

    Applicant Parameters are used to narrow the search results accordingly. Each time the user searches using the search parameters, the applicant board will change each time, according to the filters that was used. It is important to take note that when the filters are used, the page will not reload, instead, a loader will show which indicates that the system is loading the data.

    Below are the search parameters used for the Applicant Finder.

    1. "With Experience" - Narrows down the applicants with saved Applicant experiences.

    2. "Search by Skill" - The applicant will be narrowed according to their saved skill. So long as their skill is/ contains the text typed, the Applicant finder will accept it as true.

    3. "Select Qualification" - This refers to an applicant's educational attainment.

    4. "Select Field of Study" - Applicants are narrowed down according to the field of their studies.

    5. "Salary Range" - Refers to the salary expected by applicants. By adjusting the the slider or by filling the value of the textfield, the value of the salary search parameter is determined.

  • Part II

    This is the second set of parameters used in applicant searching. Most of the parameters here are optional, but these parameters will help make your Applicant search easier. First, type the specific text in the input text box; and then, select your specific search parameter. Unlike the other parameters, you can only select one from these four parameters. The Applicant finder will narrow down the Applicant according to the typed text and the parameter used. For example, if the user typed "University" in the input field and selects "University Name" as the parameter, the Applicant finder will interpret this as: An applicant that has enrolled in a university name, containing the word "University".

    Next are the buttons. There are two buttons in the applicant finder.

    The first one is the "Search". Once this is pressed, an additional parameter will be added in the list of search parameters. (Note: This will not reset any of the previous searched parameters.).

    The second button is "Reset". This button doesn't reset the parameters entirely. It only resets the parameters that are part of Applicant Parameters 2(For more information, refer to title).

  • Applicant Card

  • Basic Information


    The Applicant card is short summary of a Applicant. It contains the Applicant summary, the company logo, like buttons and link.

    1. Applicant Profile Picture- This is the profile picture of the applicant.

    2. Applicant Summary - Contains a short self-summary made by the applicant him/her self.

    3. Link - Opens the Applicant Information window which contains complete information about the Applicant.

    4. Like/Ignore buttons - Pressing like allows employers to send a match request with employers, while the ignore button removes the applicant card from the screen.
