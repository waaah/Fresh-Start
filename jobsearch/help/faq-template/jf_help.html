  • Applicant Search Parameters

  • Part I

    Job Parameters are used to narrow the search results accordingly. Each time the user searches using the search parameters, the job board will change each time, according to the filters that was used. It is important to take note that when the filters are used, the page will not reload, instead, a loader will show which indicates that the system is loading the data.

    Below are the search parameters used for the Job Finder.

    1. "Search Region" - This is basically the region where the company resides. Once a region is selected, the cities parameters will be filled with a list of cities, according to the option that was chosen.

    2. "Search City" - During the loading of the page, this field will always be empty. The user has to select a region first and then the cities will follow. When a specific region is selected, this field will be filled with data that contains cities to that specific region.

    3. "Search Specialization" - This refers to the specific skillsets needed by the company for a certain job.

    4. "Select Employment Type" - A set of employment types. A job could either be "Full Time", "Part Time" or "Freelance". Jobs under the Any Employer type is considered as any of the three.

    5. "Salary Range" - Refers to the salary given by given by companies. Jobs are filtered down between the job maximum and minimum salary. By adjusting the the slider or by filling the value of the textfield, the value of the salary search parameter is determined.

  • Part II

    This is the second set of parameters used in job searching. Most of the parameters here are optional, but these parameters will help make your job search easier. First, type the specific text in the input text box; and then, select your specific search parameter. Unlike the other parameters, you can only select one from these four parameters. The job finder will narrow down the job according to the typed text and the parameter used. For example, if the user typed "Wanted" in the input field and selects "Job Name" as the parameter, the job finder will interpret this as: A job that has a job name, containing "Wanted".

    Next are the buttons. There are two buttons in the job finder.

    The first one is the "Search". Once this is pressed, an additional parameter will be added in the list of search parameters. (Note: This will not reset any of the previous searched parameters.).

    The second button is "Reset". This button doesn't reset the parameters entirely. It only resets the parameters that are part of Job Parameters 2(For more information, refer to title).

  • Applicant Card

  • Basic Information


    The job card is short summary of a job. It contains the job summary, the company logo, like buttons and link.

    1. The Company Logo - Official Brand of the company. This is important for making a first good. impression

    2. Job Summary - Contains information regarding the job.

    3. Link - Opens the Job Information window which contains complete information about the job.

    4. Like/Ignore buttons - Pressing like allows applicants to send a match request with employers, while the ignore button removes the job card from the screen.
